# Arduino / NodeMCU setup used by this project

## Board package

Board Manager URL:

```text
https://arduino.esp8266.com/stable/package_esp8266com_index.json
```

Installed core:

```text
esp8266 by ESP8266 Community 3.1.2
```

Selected board:

```text
NodeMCU 1.0 (ESP-12E Module)
```

Arduino CLI FQBN:

```text
esp8266:esp8266:nodemcuv2
```

## Arduino libraries installed

Install these from Library Manager:

```text
Firebase Arduino Client Library for ESP8266 and ESP32 4.4.17
LiquidCrystal_I2C 2.0.0
```

The following headers are provided by the ESP8266 core / Arduino core, so they do not need separate Library Manager installs:

```text
ESP8266WiFi.h
Wire.h
Servo.h
Arduino.h
```

## Original tested port

```text
COM7
```

Use the port that appears on the target computer.
